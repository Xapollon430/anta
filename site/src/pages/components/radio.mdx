# Radio

A single option within a [`RadioGroup`](/components/radio-group/). The group owns selection, keyboard navigation, and the form value—a `Radio` is only interactive when placed in one.

```tsx folded
<Radio value="a">Brand</Radio>              {/* brand is the default */}
<Radio value="a" tone="neutral">Neutral</Radio>
<Radio value="a" tone="info">Info</Radio>
<Radio value="a" tone="success">Success</Radio>
<Radio value="a" tone="warning">Warning</Radio>
<Radio value="a" tone="critical">Critical</Radio>
<Radio value="a" tone="#e0457b">Custom</Radio>
```

</Col>
<Col>

Six named tones — `brand` (default), `neutral`, `info`, `success`, `warning`,
`critical`. A `Radio` with no `tone` inherits its group's (`brand` by default);
set `tone` to override just that option. In addition, a custom tone is possible:
pass any literal CSS colour and the radio will adapt to it.

</Col>
</Columns>

<Disclosure title="Component tokens" open>

Override any of these on a single instance via `style`, or on a selector
wrapping the radio. The ring background is the themed `--bg-1` role; the fills,
borders, disabled overlays, and focus ring are per-tone values with a matching
dark-mode block. Dimensional values (control size, dot size, ring thickness,
label type scale) are exposed as their own tokens.

| Token | Description |
|---|---|
| `--radio-control-size` | Diameter of the ring (16 / 18 / 20px per `size`). |
| `--radio-dot-size` | Diameter of the selected centre dot. |
| `--radio-gap` | Space between the ring and the label. |
| `--radio-border-width` | Ring thickness. |
| `--radio-bg` | Unselected ring background. Defaults to the `--bg-1` role. |
| `--radio-border`<br />`--radio-border-hover`<br />`--radio-border-active` | Unselected ring outline across rest / hover / active. |
| `--radio-fill`<br />`--radio-fill-hover`<br />`--radio-fill-active` | Selected (ring + fill) across rest / hover / active. Re-pointed per `tone`. |
| `--radio-dot` | Centre-dot colour (default white). |
| `--radio-dot-disabled` | Centre-dot colour when disabled. |
| `--radio-bg-disabled`<br />`--radio-border-disabled` | Ring fill / outline when disabled. |
| `--radio-focus` | Keyboard focus-ring colour. |

</Disclosure>
